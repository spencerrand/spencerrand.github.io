Eveyone knows that the temperature gets hotter as you approach the equator, but can we use data to definitely prove it? I wrote a Python script to visualize the weather of over 500 cities across the world. The cities were chosen at random so that they are of varying distances from the equator. The script makes an API request to OpenWeatherMap and creates a representative model. There are four charts:
  1. Temperature (F) vs. Latitude
  2. Humidity (%) vs. Latitude
  3. Cloudiness (%) vs. Latitude
  4. Wind Speed (mph) vs. Latitude
